Use 'component.attachShadow' to create a Shadow DOM.
#{m1}
attachShadow: function(componentInstance, templateId) {
    var templateElement = document.querySelector(templateId)
    var clone = document.importNode(templateElement.content, true);
    var shadow = componentInstance.attachShadow({mode: 'open'})
    shadow.appendChild(clone)
    return shadow
}

Know the Component lifecycle.
#{m2}
  1. Use the Component in page HTML (e.g. <gauge-el>)
  2. Import or fetch (e.g. /GaugeEl.html) and add to DOM (appendChild(...))
  3. Register the Component (document.registerElement(...), only once)
  4. Attach Shadow DOM in Component 'createdCallback' and trigger custom rendering
